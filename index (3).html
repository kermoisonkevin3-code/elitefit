🍪 Biscuits & Gâteaux Zéro Calorie — Guide
Options validées : Fibre One, Gerlinea, ou recettes maison (farine d'amande + édulcorant + poudre de protéine). Attention : même "zéro sucre", certains ont des graisses. Lis les étiquettes — vise moins de 50 kcal/portion.
⚠️ "Zéro sucre" ≠ Zéro calorie. Les édulcorants ne cassent pas le jeûne sur les calories mais peuvent entretenir l'appétit pour le sucre. Essaie de limiter à 1-2 portions par jour.

PROGRAMMES COMPLETS
Plan alimentaire + sportif sur 12 semaines — Objectif Armée de Terre
Feuille de Route — 12 Mois
Phase 1 : Mois 1–3 (Fondations)
  • ⚖️ Objectif : -13 kg (113 kg)
  • 🍽️ Calories : 1650 kcal/jour
  • ⏱ Jeûne : 16:8 quotidien
  • 💪 Sport : 3 séances/semaine
  • 🚶 Marche : 30min/jour minimum
  • 📋 Focus : Installer les habitudes, apprendre à cuisiner sainement
Phase 2 : Mois 4–6 (Montée en Charge)
  • ⚖️ Objectif : -13 kg supplémentaires (100 kg)
  • 🍽️ Calories : 1500 kcal/jour
  • ⏱ Jeûne : 16:8 + 1x 24h/semaine
  • 💪 Sport : 4-5 séances/semaine
  • 🏃 Cardio : Course lente 20min
  • 📋 Focus : Endurance, force, premiers tests militaires
Phase 3 : Mois 7–9 (Intensification)
  • ⚖️ Objectif : -12 kg (88 kg)
  • 🍽️ Calories : 1400 kcal/jour
  • ⏱ Jeûne : 16:8 + 2x 24h/semaine
  • 💪 Sport : 5 séances/semaine
  • 🏃 Cardio : Course 30min
  • 📋 Focus : Préparation physique militaire, HIIT intensif
Phase 4 : Mois 10–12 (Objectif Final)
  • ⚖️ Objectif : -13 kg → 75 kg ✅
  • 🍽️ Calories : 1500 kcal/jour (maintien/rééquilibrage)
  • ⏱ Jeûne : Flexible
  • 💪 Sport : 5 séances militaires/semaine
  • 🏃 Cardio : Course 5km en moins de 25min
  • 📋 Focus : Tests d'aptitude, préparation concours
